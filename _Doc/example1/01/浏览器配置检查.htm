浏览器配置检查

    

为了可以正常使用这个学习系统，您的浏览器需要：

  1. 是 Microsoft Internet Explorer 6.0 或以上版本
  2. 允许 Javascript
  3. 允许 Cookies
  4. 允许弹出窗口
  5. 安装有 Macromedia Flash播放器 7.0 或以上版本
  6. 安装有 Sun Java 1.4 或以上版本的Java插件(可选)

我们正在检查您的浏览器以确定是否满足以上的要求。如果看到任何一项失败！，请按照提示操作。如果每一项都是成功！，您可以直接开始使用此学习系统。

如果上面有任何一项失败！，请看以下部分。

  1. 浏览器
    请通过Internet Explorer去使用此学习系统。
    如果您使用的是旧版本的Internet Explorer，请到    Windows Update更新。

    再次检查

  2. Javascript
    如果被禁用了，请根据以下步骤启用：
    1. 打开浏览器的 工具 > Internet 选项 > 安全 > Internet > 自定义级别
    2. 找到“脚本”部分，如下图
    3. 确定“活动脚本”是“启用”状态

    再次检查

  3. Cookies
    如果被禁用了，请根据以下步骤启用：
    1. 打开浏览器的 工具 > Internet 选项 > 隐私 > 高级
    2. 确定“覆盖自动 cookie 处理”是没有选上，如下图

    再次检查

  4. 弹出窗口
    如果您的浏览器禁止任何弹出窗口，请把它取消。或者您安装了任何可以拦截弹出窗口的工具，请把该功能也取消。比如，您使用的是Google工具栏，请点击以下按钮

    如果是以下情况，代表当前是允许弹出窗口


    再次检查

  5. Flash播放器
    如果您使用的是旧版本的Flash播放器，或者是没有Flash播放器，请到    Macromedia Download Site下载并安装。

    再次检查

  6. 安装Java插件
    1. 首先，检查是否已经安装Java插件：
      1. 打开浏览器的 工具 > Internet 选项 > 高级
      2. 检查 "java(Sun)" 部分是否存在，如下图
      3. 如果 "java(Sun)" 部分如果存在，请使其选项被选上。如不存在，请根据下面步骤 b. 安装Java插件
      4. 如果 "java(Sun)" 部分下面的选项被选上，请取消该选项并重新启动浏览器。再次进入学习系统，检查是否需要安装Java插件
    2. 如果没有Java插件，请根据以下步骤安装：
      （请注意，您必须拥有操作系统的管理员权限才能进行安装）
      1. 如果您的操作系统是32位的请下载下载Java插件(jre1.6 32位, 15.7 MB) ; 如果您的操作系统是64位的请下载下载Java插件(jre1.6 64位, 15.6 MB)
      2. 安装Java插件
      3. 重新启动浏览器并进入学习系统

    再次检查

  7. 更新Java插件
    如果使用的是旧版本的Java插件，请根据以下步骤更新：
    （请注意，您必须拥有操作系统的管理员权限才能进行安装）
    1. 如果您的操作系统是32位的请下载下载Java插件(jre1.6 32位, 15.7 MB) ; 如果您的操作系统是64位的请下载下载Java插件(jre1.6 64位, 15.6 MB)
    2. 安装新的Java插件
    3. 重新启动浏览器并进入学习系统

    再次检查
